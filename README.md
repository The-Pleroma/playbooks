# The Pleroma — Playbooks

Shared playbooks, manuals, and guides for the crew. Each member maintains their own docs and contributes to shared knowledge.

## Structure

```
gsd/              # GSD (Get Shit Done) workflow guides
  ubuntu.md       # Ubuntu/Linux setup + GSD workflow
  android.md      # Android dev environment + GSD workflow
platform/         # Platform-specific setup guides
members/          # Per-member logs, notes, personal playbooks
  seph/
```

## How to Use

1. Clone this repo
2. Add your playbooks to `members/<your-name>/` or to a shared topic folder
3. Commit and push — no PRs required, this is a living knowledge base

## Contributing

- Drop guides in the relevant folder
- Keep docs practical — commands over theory
- Update existing docs when things change rather than creating new versions
